# Your Wall Express TypeScript

Este repositorio contiene el backend de Your Wall, donde se utiliza:

* Express
* Mongoose
* TypeScript

Éste repositorio forma parte del artículo explicado en DEV.to:

## [Post sobre Vuex con TypeScript](https://dev.to/rodrigoherera/vuex-con-typescript-362j-temp-slug-8714324?preview=cdc43bcff9e52ee09b24df68960539e347bba2435a4954e860bd549f2a5e01b11b56b9fba0c5a0e54b8e4ab0cd282394631b0c62b85db1bdb5a310ce)

## Instalación

Para instalar las dependencias:

```cmd
npm install
```

Para correr el proyecto:

```cmd
npm run dev
```

Para compilar el proyecto:

```cmd
npm run build
```
